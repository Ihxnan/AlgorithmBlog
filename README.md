## 编译运行

### 基本编译命令
```bash
# 编译单个文件
g++ filename.cpp
```

### 推荐编译选项
```bash
g++ -std=c++17 -O2 -Wall filename.cpp -o filename
```
- `-std=c++17`：使用 C++17 标准
- `-O2`：优化级别 2，提高运行效率
- `-Wall`：显示所有警告信息（调试时推荐）

### 快速编译运行
```bash
# 临时编译运行（自动清理可执行文件）
g++ filename.cpp && ./a.out && rm a.out
```
